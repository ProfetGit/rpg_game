RPG Game Documentation

Introduction

Welcome to the RPG Game documentation. This is a text-based role-playing game featuring character creation, branching storylines, and dynamic gameplay based on character stats.

Architecture

The game is built around three main components:

  • Character: Handles character creation and stat management
  • Game: Manages game state and core gameplay mechanics
  • Story: Controls the branching narrative and story progression

Key Features

  • Character Classes: Choose between Warrior, Scholar, Diplomat, or Rogue
  • Stat-Based Choices: Story decisions are influenced by character attributes
  • Inventory System: Collect and manage items throughout your journey
  • Multiple Chapters: Experience a rich story across three chapters
  • Save/Load System: Save your progress and continue later

Getting Started

  1. Initialize the game using initializeGame()
  2. Create a character with createCharacter()
  3. Start the story with initializeStory()
  4. Use displayCurrentScene() and processPlayerChoice() for gameplay

Dependencies

  • ncurses: Required for terminal-based UI

Building the Project

Use the provided Makefile:

make clean
make
